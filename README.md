# Praktika TI

Ky repository përmban detyrën praktike për lëndën e Teknologjisë së Informacionit.

## Përmbajtja

- Dokumenti Word i detyrës së plotësuar
- Përshkrimi i modeleve të aplikacioneve web
- Strukturimi i funksionaliteteve kryesore
- Teknologjitë e sugjeruara: HTML, CSS, JavaScript, PHP/Python dhe MySQL
